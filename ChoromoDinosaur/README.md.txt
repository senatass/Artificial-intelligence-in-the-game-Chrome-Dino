*Chrome Dino Game (Python - Pygame)

This project is a similar version of the classic Google Chrome Dino Run game, designed to be used with artificial intelligence, and it is still under development. 
It was developed using Python (Pygame).

*Features
1) Jumping, ducking, and running animations.
2) Small and large cactus obstacles.
3) Flying birds (at different heights)
4) Randomly moving clouds.
5) Endless running mechanics.
6) Increasing difficulty with speed.
7) Score system.

*Before running the project, the following software must be installed on your system:
1) Python
2) Pygame
